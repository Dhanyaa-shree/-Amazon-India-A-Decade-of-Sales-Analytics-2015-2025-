# 🎯 Problem Statement

Develop a comprehensive E-Commerce Analytics solution using Amazon India's transactional data spanning 2015–2025 to transform raw and inconsistent data into actionable business intelligence dashboards.

The project focuses on:

* 🧹 Cleaning and preprocessing large-scale e-commerce data with real-world quality issues
* 📊 Performing Exploratory Data Analysis (EDA) to uncover business patterns and trends
* 🗄️ Integrating cleaned data into SQL databases for efficient querying and reporting
* 📈 Building interactive Power BI dashboards for executive and operational decision-making
* 💡 Delivering actionable insights for revenue growth, customer retention, product performance, and operational efficiency

# 🏗️ Project Approach

## 1️⃣ Data Understanding

* Analyzed Amazon India transaction data spanning 10 years
* Studied customer behavior, product catalog, payment methods, and operational metrics
* Identified data quality issues requiring preprocessing

## 2️⃣ Data Cleaning & Preparation

* Handled missing values and inconsistent formats
* Standardized date, category, payment, and customer fields
* Removed duplicates and corrected outliers
* Created a clean dataset suitable for analytics and reporting

## 3️⃣ Exploratory Data Analysis (EDA)

Performed extensive EDA using Python libraries:

* Revenue trend analysis
* Seasonal sales patterns
* Customer segmentation
* Product category analysis
* Brand performance evaluation
* Payment behavior analysis
* Return and cancellation analysis

## 4️⃣ SQL Integration

* Designed relational database schema
* Loaded cleaned datasets into SQL
* Created queries for KPI generation and reporting
* Connected SQL outputs with Power BI

## 5️⃣ Power BI Dashboard Development

Built a multi-page interactive dashboard with:

* KPI Cards
* Dynamic Filters & Slicers
* Page Navigation

# 📈 Key Business Insights

* Smartphones generated the highest revenue contribution across all categories.
* Samsung emerged as the leading brand based on market share.
* Prime customers contributed significantly more revenue than non-prime customers.
* December recorded the highest revenue due to seasonal and festive demand.
* Delivered orders accounted for more than 90% of total transactions.
* Customer churn remained relatively low, indicating strong customer retention.
* UPI and digital payments showed increasing adoption over the study period.
* Higher-rated products demonstrated stronger sales performance and lower return rates.
